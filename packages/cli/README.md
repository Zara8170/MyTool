# mytool-ai

Claude Code observability CLI. Track token usage, tool calls, and sessions across your AI coding sessions.

## Install

```bash
npm install -g mytool-ai
```

## Usage

```bash
cd /path/to/your/project
mytool
```

The CLI will guide you through linking your project and installing Claude Code hooks.

## Commands

```bash
mytool          # Setup / link project
mytool status   # Show auth and project status
mytool logout   # Log out
```
